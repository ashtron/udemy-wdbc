RESTful Routes

Name Path HTTP Verb Purpose Mongoose Method
index /things GET list all things Thing.find()
new /things/new GET show new thing form
create /things POST create new thing, then redirect somewhere Thing.create()
show /things/:id GET show information about a particular thing Thing.findById()
edit /things/:id/edit GET show edit form for a particular thing Thing.findById()
update /things/:id PUT update a particular thing, then redirect somewhere Thing.findByIdAndUpdate()
destroy /things/:id DELETE delete a particular thing, then redirect somewhere Thing.findByIdAndRemove()
